More than just a To-Do App.

In a world of complex and expensive productivity tools, we wanted to create something different: a simple, beautiful, and free tool that just works. This project started as a learning journey and a passion for building useful things.

Our philosophy is simple:

  • check Focus on Essentials: We only include features that you actually need, making the app fast and easy to use.
  • check Open & Transparent: The code is open-source. You can see exactly how it works and be sure your data is safe.
  • check Free Forever: The core functionality will always be free for everyone.

A One-Person Team

Hi, I'm a passionate developer building this app in my free time. I love creating tools that solve real-world problems and exploring new technologies like Angular and Firebase. This project is my way of giving back to the community and honing my skills.
